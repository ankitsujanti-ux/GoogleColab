# Hornets × Microsoft Fabric — Interactive Demo

An interactive, self-contained dashboard demo showing how Microsoft Fabric complements
Databricks + Power BI for the Charlotte Hornets (Fan 360, Ticketing & Revenue, and a live
Smart Venue command center). All figures are simulated sample data for illustration.

## How to view it

**Option A — one file, no setup (recommended for sharing):**
1. Open `hornets-demo-standalone.html` in this folder.
2. Click **Download raw file** (top-right of the file view on GitHub).
3. Double-click the downloaded file — it opens in any browser and works fully offline.
   Everything (logo, charts, live simulation) is embedded in that single file.

**Option B — run the folder locally:**
1. Clone the repo.
2. Open `index.html` (uses `apexon-logo.png` alongside it).

## Files
- `hornets-demo-standalone.html` — the entire demo in one portable file (share this).
- `index.html` — multi-file version.
- `apexon-logo.png` — logo used by `index.html`.
